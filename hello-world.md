### Merhaba

1. **Başlıklar**:
   - `# Başlık 1` (H1)
   - `## Başlık 2` (H2)
   - `### Başlık 3` (H3)
   
2. **Kalın Metin**:
   - `**kalın metin**` veya `__kalın metin__`
   
3. **İtalik Metin**:
   - `*italik metin*` veya `_italik metin_`
   
4. **Liste**:
   - Sıralı Liste: 
     - `1. Birinci öğe`
     - `2. İkinci öğe`
   - Sırasız Liste: 
     - `- Birinci öğe`
     - `* İkinci öğe`

5. **Bağlantılar**:
   - `[metin](URL)`

6. **Görüntü**:
   - `![alternatif metin](görüntü URL'si)`

7. **Kod**:
   - Tek satırlık kod: `` `kod` ``
   - Çok satırlı kod:
     ```
     ```
     kod
     ```
     ```

8. **Alıntı**:
   - `> Alıntı metni`

9. **Yatay Çizgi**:
   - `---` veya `***`

Bu komutlar, Markdown formatında metin yazarken işinize yarayacaktır!
